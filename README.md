# galvinlam.github.io

GitHub Pages portfolio site for Galvin.

## Publish

1. Create the repository `galvinlam.github.io` under the `galvinlam` GitHub account.
2. Push this folder to the `main` branch.
3. In GitHub repo settings, enable Pages from `main` and `/ (root)` if it does not auto-enable.

Published URL:

- `https://galvinlam.github.io`
